# Sanduqkin

Secure family information organizer for iOS and Android.

Start here:

- Product/build source of truth: `Vault_BRD_v1.0.md`
- Session continuity and progress tracking: `HANDOFF.md`
- Mobile app: `apps/mobile`
- Account deletion operations: `docs/account-deletion-operations.md`

Run the mobile app:

```bash
npm install
npm run mobile
```
